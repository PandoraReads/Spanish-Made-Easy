# Spanish Made Easy (SME) - Obsidian Spanish Learning Assistant

**Spanish Made Easy (SME)** is a comprehensive Spanish learning plugin designed specifically for Obsidian. It integrates a dictionary assistant, FSRS flashcard memory system, and video shadowing features to provide you with the smoothest immersive Spanish learning experience possible.

---

## 🚀 Core Features

### 1. 📖 Smart Dictionary Assistant
*   **Multiple Dictionary Support**: Integrates Google Translate (Spanish-English), Bing, and supports local offline MDX dictionaries.
*   **Lemmatization**: Supports precise identification of Spanish verb conjugations, noun plurals, and adjective gender/number changes, directly displaying the word's root form.
*   **One-Click Card Creation**: Click "Add to Vocabulary" in dictionary results to automatically capture current context and pronunciation, saving to your review library.
*   **Spanish Pronunciation**: Integrated Google TTS providing authentic Spanish native speaker pronunciation.

### 2. 🧠 FSRS Flashcard Review
*   **Scientific Memory**: Built-in advanced FSRS (Free Spaced Repetition Scheduler) algorithm that intelligently calculates review intervals.
*   **Immersive Experience**: Beautiful card styles with typing practice, auto-pronunciation, and example sentence display.
*   **Note Sync**: Two-way synchronization between flashcards and your Obsidian notes.

### 3. 🎬 Video Shadowing Workshop
*   **Immersive Shadowing**: Supports YouTube and local videos with automatic timestamp extraction.
*   **Dictation Practice**: Built-in dictation mode with instant accuracy verification.
*   **Speed Adjustment**: Supports 0.5x - 2.0x speed control to master every Spanish linking detail.

### 4. ✨ Highlight Notes
*   **Smart Highlighting**: Highlight text in your notes with color-coded tags.
*   **AI Enhancement**: Support for DeepSeek and Gemini AI to automatically generate translations, grammar analysis, and contextual explanations.
*   **Card Management**: Organize and review all your highlighted snippets in a beautiful card gallery interface.

## 📥 Installation Guide

1.  **Manual Installation:**

## 🛠️ Configuration

### Language Settings
SME now supports both **Chinese** and **English** interfaces:
1.  Go to **Settings → Spanish Made Easy**
2.  Select your preferred language from the "Language / 语言" dropdown
3.  Reload Obsidian to apply changes

### Dictionary Setup
*   **Default Service**: Google Translate (Spanish-English) - Free, no configuration required
*   **Local Dictionaries**: Supports MDX format dictionaries for offline use
    *   Set absolute paths to your `.mdx` and optional `.mdd` files in settings
    *   Supports up to two dictionary sets for comprehensive results

### Vocabulary Notebook
*   **Default Folder**: Root of your vault
*   **Default Filename**: `Vocabulary`
*   Customize both in plugin settings to match your workflow

### AI Integration (Optional)
*   Configure your preferred AI provider (DeepSeek, Gemini, or Ark)
*   Enter your API key in settings
*   Use AI features in highlight notes for instant translations and grammar analysis

## 📖 Usage Guide

### Looking Up Words
1.  **Text Selection**: Select any Spanish word in your notes
2.  **Keyboard Shortcut**: Use `Ctrl/Cmd + K` (or click the search icon that appears)
3.  **Dictionary Panel**: View definitions, pronunciation, and examples
4.  **Add to Vocabulary**: Click the "Add to Vocabulary" button to save for later review

### Flashcard Review
1.  Click the flashcard icon in the sidebar or use the command palette
2.  Review today's due cards with FSRS algorithm
3.  Rate each card: Forgot, Hard, Good, or Easy
4.  Track your progress with the learning heatmap

### Video Shadowing
1.  Open a note containing a YouTube link or embedded video
2.  Add timestamped subtitles in format: `[00:12] Your text here`
3.  Open the Shadowing Workshop view
4.  Practice shadowing or dictation with automatic synchronization

### Highlight Notes
1.  Select text in any note
2.  Choose a color (yellow, pink, blue, or green)
3.  Add personal notes or use AI for translations/analysis
4.  Review all highlights in the Card Manager view
